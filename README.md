> **Honest note** — you can't run all tracks at full speed at once. Foundation is shared by everyone, so build that first no matter what. After that, pick one primary domain and treat the rest as exploration/backup. Given the DSA + Linux work you'd need for SDE anyway, and how it overlaps with Cybersecurity, that pairing gets you to a job fastest — but explore all domains below and choose what actually pulls you.

---

## 📌 Phase 0 — Foundation (Everyone starts here)
**Month 1–4 · Jul–Oct 2026**

- [ ] Complete CS50x (Harvard) fully, including the final project — covers C basics (variables, loops, functions, arrays, pointers) plus Python, SQL, and web dev
  > **Not into CS50?** Equally solid options: **freeCodeCamp** (free, project-based, very beginner-friendly), **The Odin Project** (structured, community-backed, good if you lean web dev), or **Python Crash Course** book + **CS50P** if you want Python-first instead of C-first. Whichever you pick — finish it fully with a final project, don't just watch.
- [ ] Transition C to C++: learn OOP (classes, objects) and STL (vectors, maps, sets) — needed later for DSA/competitive coding
  > CS50 teaches plain C, not C++. This is a quick 1–2 week add-on once CS50 is done, not a from-scratch language.
- [ ] Learn Git & GitHub: commits, branches, PRs, resolving conflicts
- [ ] Get comfortable in Linux terminal: file system, permissions, piping, shell basics
- [ ] Discrete math refresher: logic, sets, basic combinatorics
- [ ] Push typing speed to 40+ WPM
- [ ] Start daily spoken-English practice (shadowing, self-recording) — treat it as seriously as code
